Simple quiz widget, redirect the website visitor to another page when they choose the right answer.

== Description ==
Simple quiz widget, one question and three answers when the website visitor chooses the right answer action button will show-up to redirect him / her to another page to collect free gift, discount or anything else you offer them.

= Plugin website =
[am-plugins.com](https://am-plugins.com/am-action-quiz) is the official homepage for this plugin.

== Installation ==
= From your WordPress dashboard =

1.	Visit 'Plugins > Add New'.
2.	Search for 'AM-Action quiz'.
3.	Install AM-Action Quiz plugin.
4.	Activate the plugin through the 'Plugins' menu in WordPress

= From WordPress.org =
1.	Download and unzip AM-Action quiz plugin.
2.	Upload the am-action-quiz folder to the /wp-content/plugins/ directory
3.	Activate the plugin through the 'Plugins' menu in WordPress
